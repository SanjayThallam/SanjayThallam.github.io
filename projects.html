Projects

I enjoy learning new skills by doing hands on projects
A small selection of the projects I have worked on is listed below

IOT Plant Management System

September 2021 - Present

Developing an MSP430 based remote plant status monitoring system. I am currently developing a break out PCB for all of the sensors used for status monitoring as well as a wifi module. The next steps will be to link with AWS Services to upload data which can be accessed using a mobile application.

Spiking Neural Network Hardware Accelerator

September 2021 - Present

Currently in preliminary stages of developing a spiking neural network hardware accelerator using Chipyard and RocketChip. In this project, I specifically am performing research into potential implementations and areas of optimization. In the near future, I will be designing a spiking core for the accelerator.

RISC-V Softcore

January 2021 - May 2021

Implemented a 32-bit RISC-V Datapath on an Intel-Alterra MAX10 FPGA using Verilog. Utilized ModelSim for RTL analysis and to ensure instructions were processed properly. The softcore is able to execute base the RV32I instruction set.

Market+

March 2021

Built for ProfHacks 2021. Built an application in Scratch (Yes, Scratch) to connect food vendors in developing nation markets with potential local customers. The application idea included giving the customer the ability to price shop multiple vendors to save money and to give vendors a 1 stop shop for store management.

Rowan Formula Racing - FSAE Electric

September 2019 - May 2021

Was a member of the ECE subteam for Rowan University's Formula SAE Electric race team. Worked on Electrical System Testing, Regenerative Braking System Code (CAN), Failure Mode and Effects Analysis, Parts Procurement and Cooling System Design

Arduino Sumo Robot

December 2019

Designed, Programmed an Assembled an Arduino based Sumo robot. Implemented code for enemy detection, out-of-bounds avoidance and path planning. Won first place out of the freshman who participated
